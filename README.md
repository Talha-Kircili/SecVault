# Sec⚡Vault

CLI Passwort Manager for Linux.

# Features

- Portable

- Simple CLI

- Password Recovery

- AES Encrypted Storage

- Passwort Generator

- Breach Check

# Install

    sed 's/#.*//' requirements.txt | xargs sudo apt-get -y install

# Compile

    make

# Usage

    ./secvault
